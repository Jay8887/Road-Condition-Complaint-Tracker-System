# Road-Condition-Complaint-Tracker-System
A full-stack web application that allows users to report road problems and track the status of their complaints.

## Features

* Report road damage or issues
* Track complaint status
* Admin can manage complaints
* User-friendly interface

## Technologies Used

* Java
* Spring Boot
* MySQL
* HTML, CSS, JavaScript
